# ArrayToolkit (C语言数组工具库)

这是一个基于标准 C 语言开发的数组与矩阵处理工具库，旨在提供高效、简洁的数组操作接口。

## 🚀 项目特点
* **内存安全**：统一的动态内存分配与释放管理。
* **功能丰富**：支持一维/二维数组的创建、初始化、获取元素及基本运算。
* **结构标准**：采用函数库（Library）与测试程序（TestApp）分离的工业级工程架构。

## 🛠️ 开发环境
* **操作系统**: Windows 11
* **编译器**: MSVC (Visual Studio 2022)
* **版本控制**: Git

## 📂 目录结构说明
* `/ArrayToolkit`: 核心库源码，包含数组处理的所有底层实现。
* `/TestApp`: 测试工程，包含 `main` 函数，用于验证库函数的功能。
* `ArrayToolkit.sln`: Visual Studio 总解决方案文件。

## 📝 如何使用
1. 使用 Visual Studio 打开 `ArrayToolkit.sln`。
2. 将 `TestApp` 项目设为“启动项目”。
3. 按 `Ctrl + F5` 运行测试程序，查看控制台输出。
